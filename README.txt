# Portfolio Site

Тёмный Bootstrap-сайт для публикации твоих версток.

## Как использовать
1. Открой `index.html`
2. Измени карточки проектов на свои
3. Для каждого проекта создай папку внутри `projects`
4. Внутрь положи свой `index.html`

## Пример структуры
projects/
└── my-new-layout/
    └── index.html
